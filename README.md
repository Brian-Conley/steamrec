# steamrec

## REQUIREMENTS
### Linux
- Docker
- Docker Compose
### Windows
- Docker Desktop (comes with Docker/Docker Compose)  
Download from www.docker.com/products/docker-desktop/
- WSL2  
To install WSL2:
1. Open PowerShell
2. Run: 
```batch
wsl --install
```

## RUNNING THE PROJECT
1. **Build and start containers:**
```bash
docker compose up --build
```
